Знакомство!

При создании веб-страниц используются два языка: HTML и CSS. HTML отвечает за структуру и содержание, а CSS — за оформление. Браузер объединяет HTML- и CSS-код и формирует внешний вид страницы.

HTML теги всегда заключены в < >. Можно выделить одиночные и парные теги

  • <p>Абзац</p>
  • <h1>Заголовок первого уровня</h1>
  • <h2>Заголовок второго уровня</h2>
  • <h3>Заголовок третьего уровня</h3>
  • <h4>Заголовок четвертого уровня</h4>
  • <a href="адрес">Гиперссылка</a>
  • <em>Курсив</em>
  • <strong>Жирность</strong>
  • <hr>Горизонтальная линия
  • <img src="адрес">Рисунок

Теги могут иметь атрибуты. Общая модель выглядит примерно так:

 <имя-тега атрибут1="значение1" атрибут2="значение2" ...>

Для того, чтобы понять какие именно атрибуты имеет тег существует такая шпаргалка

CSS состоит из множества правил, примерно таких:

селектор{свойство1: значение;свойство2: значение;}
em{color:green;}

При стилизации абзаца с помощью внешних стилей все они получились одинакового цвета. Нужно было сделать следующее: добавить абзацам разные классы, а в CSS использовать селекторы не по тегу, а по классам. Класс — это всего лишь один из атрибутов HTML-тегов, например:

  • <p class="important">...</p>
  • <p class="help">...</p>

В CSS можно задавать стили только для элементов с определённым классом. Для этого используется селектор по классу, который пишется так .имя-класса, например:

  • .important { color: red; } — выберет все теги с классом "important"
  • .help { color: green; } — выберет все теги с классом "help"
